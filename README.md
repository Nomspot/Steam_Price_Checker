# Steam_Price_Checker

A simple Python automation project using Selenium -> script automaiton to check prices of a steam game based on the store id

## What it does
- Opens a steam in headless (invisable) mode
- Checks page content to find the prices of all availble editions
- Prints it back to you

## Technologies
- Python
- Selenium

## How to run
1. Install requirements :

    `pip install threading`
    
    `pip install selenium`
2. Run main.py
